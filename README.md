# 📱 TOEIC Vocabulary Master (多益單字大師)

這是一個使用 SwiftUI 開發的 iOS 學習 App，旨在幫助使用者利用零碎時間背誦多益單字。

## ✨ 主要功能
* **📚 單字卡學習**：支援 3D 翻轉卡片、發音、分類篩選。
* **📝 隨堂測驗**：自動出題，答對答錯即時回饋，並有音效與震動。
* **❌ 錯題本系統**：自動記錄答錯的單字，直到熟記為止。
* **🔔 每日提醒**：可自訂時間的推播通知，養成學習習慣。
* **📊 進度儀表板**：首頁顯示每日目標達成率。

## 🛠️ 技術堆疊
* SwiftUI / NavigationStack
* AVFoundation (發音) / AudioToolbox (音效)
* UserNotifications (推播)
* UserDefaults / AppStorage (資料持久化)
